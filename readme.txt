== Installation ==

1. Upload the topper-pack folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure widgets, extensions, and API settings via the Topper Pack dashboard.
4. Use the Template Library or drag-and-drop widgets to start building.

== Import/Export & Reset Tools ==
Manage your plugin configuration safely and efficiently:

- **Export Settings:** – Backup all plugin settings (widgets, extensions, CPTs, API settings, and more) into a downloadable JSON file.
- **Import Settings:** – Restore saved configurations by uploading a JSON file.
- **Reset Settings:** – Reset all plugin options to default values with a confirmation step. This includes all widgets, extensions, API keys, and layout settings.
